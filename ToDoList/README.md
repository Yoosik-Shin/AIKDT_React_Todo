# 리액트 프로젝트 생성
1. 프로젝트 생성
   (CRA) - npx create-react-app todo-app
                    or
    (Vite) - create-vite todo-app --template react

2. 모듈 설치
    - npm install

3. 서버 실행
    - npm start (CRA)
    - npm run dev (Vite)

4. Components
    - Todo
        - Container.jsx
        - Header.jsx
        - Input.jsx
        - List.jsx
        - Card.jsx
        - Footer.jsx
